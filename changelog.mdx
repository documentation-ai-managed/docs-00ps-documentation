## Recent Releases

<Update label="2024-10-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Added AI-powered semantic search endpoint `/v2/ai-search` for natural language queries.
- Introduced real-time suggestion API with latency `<50ms`.
- New personalization engine using user history for tailored results.

## Breaking Changes

- Deprecated `/v1/search` in favor of `/v2/search`. Update your calls immediately.

## Improvements

- Enhanced mobile responsiveness for online services dashboard.

</Update>

## Upgrading Your Integration

Follow these steps to apply the latest changes to your application.

<Steps>
  <Step title="Review Breaking Changes" icon="alert-triangle">

    Check the `v2.0.0` release notes above for deprecated endpoints.

  </Step>
  <Step title="Update Dependencies" icon="download">

    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm install baidu-search-api@latest
        ```
      </Tab>
      <Tab title="Python" icon="python">
        ```bash
        pip install baidu-search==2.0.0
        ```
      </Tab>
    </Tabs>

  </Step>
  <Step title="Test New Endpoints" icon="play-circle">

    Use the example below to verify AI search.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const response = await fetch('https://api.baidu.com/v2/ai-search?q=natural language query', {
        headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` }
      });
      const data = await response.json();
      console.log(data.results);
      ```

      ```python
      import requests
      response = requests.get('https://api.baidu.com/v2/ai-search', 
                             params={'q': 'natural language query'},
                             headers={'Authorization': f'Bearer {YOUR_API_KEY}'})
      data = response.json()
      print(data['results'])
      ```
    </CodeGroup>

  </Step>
  <Step title="Deploy and Monitor" icon="rocket">

    Monitor your dashboard for any issues post-deployment.

  </Step>
</Steps>
