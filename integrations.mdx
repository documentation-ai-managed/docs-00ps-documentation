## Quickstart: Integrate Search API

Follow these steps to add Baidu Search to your app.

<Steps>
  <Step title="Get API Key" icon="key">
    Sign up at the [Baidu Developer Console](https://console.baidu.com) and create a new project. Copy your `{API_KEY}`.
  </Step>
  <Step title="Install SDK" icon="download">
    Choose your language and install the official SDK.
  </Step>
  <Step title="Make First Request" icon="play">
    Send a search query and handle the response.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
const response = await fetch('https://api.baidu.com/v1/search', {
  method: 'POST',
  headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
  body: JSON.stringify({ query: 'machine learning' })
});
const data = await response.json();
```
```python
import requests

response = requests.post(
    'https://api.baidu.com/v1/search',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'},
    json={'query': 'machine learning'}
)
data = response.json()
```
</CodeGroup>

## Best Practices

<Callout kind="alert">
  Always validate `{API_KEY}` server-side and use HTTPS for all requests.
</Callout>

- Rotate keys regularly.
- Implement retry logic for transient errors.
- Cache frequent search results to reduce API calls.

```javascript
// Example: Retry with exponential backoff
async function searchWithRetry(query, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      const response = await fetch('https://api.baidu.com/v1/search', {
        // ... request config
      });
      return await response.json();
    } catch (error) {
      if (i === maxRetries - 1) throw error;
      await new Promise(resolve => setTimeout(resolve, 2 ** i * 1000));
    }
  }
}
```
