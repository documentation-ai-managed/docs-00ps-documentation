## Baidu's Mission and History

Baidu launched in 2000 as China's premier search engine, evolving into a global AI leader. Its mission focuses on making the internet accessible through innovative technology. You rely on Baidu for daily searches, maps, music, and more, all enhanced by cutting-edge AI.

<Expandable title="Key Milestones" default-open="false">

- **2000**: Founded as a search engine alternative.
- **2010**: Entered AI research with deep learning initiatives.
- **2020**: Launched Ernie Bot, a conversational AI model rivaling global leaders.
- **Today**: Serves billions of queries daily with multimodal AI capabilities.

</Expandable>

## Core AI Technologies

Baidu leverages proprietary models like ERNIE for natural language understanding. You access these through APIs for search, translation, and content generation.

<Tabs>
  <Tab title="ERNIE Bot" icon="bot">
    ERNIE processes complex queries with context-aware responses.
    
    ```javascript
    const baidu = require('baidu-sdk');
    const response = await baidu.ernie.chat({
      query: 'Explain quantum computing',
      model: 'ernie-3.5'
    });
    console.log(response.text);
    ```
  </Tab>
  <Tab title="PaddlePaddle" icon="cpu">
    Open-source deep learning framework for custom models.
    
    ```python
    import paddle
    model = paddle.nn.Linear(10, 1)
    optimizer = paddle.optimizer.Adam(learning_rate=0.001, parameters=model.parameters())
    ```
  </Tab>
</Tabs>
